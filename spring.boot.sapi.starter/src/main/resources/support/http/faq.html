SAPI是什么？

SAPI是一个基于Spring的帮助开发人员扫描Restful接口的小工具，只需要一行配置，就可以看到相应的接口以及参数。


SAPI如何测试？

打开了http://127.0.0.1:{your.port}/{your.contextPath}/spai后，在每个接口右侧页面上可以看到一个绿色的三角按钮。对相关参数进行赋值后点击"Send"就可以看到相关的输出结果了。


如何获取源码or提Issues？

Sapi是一个开源的组件，任何人都可以在Apache License 2.0的规范下修改源码。https://github.com/xiaour/spring.boot.sapi.starter,同时也欢迎各位开发者Star ^_^
